# zBlog
zblog 博客

项目背景：zblog博客项目是基于前端分离形式开发的个人博客项目，项目后端采用Spring Boot技术，后期考虑增加Spring Cloud部分技术，前端采用了Vue + Element-UI框架组件搭建，目前项目处于开发中阶段。
